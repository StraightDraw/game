Police versus Guerillas#

We will investigate a genre of games called Police versus Guerrillas. The idea is that an insurgency (the guerrilla force) is striking against the government (the police). The police have two arsenals filled with weapons and amunition, and the guerrillas would like to take either one of them, or both, if possible.

Units

The police have \(p\) units they can send to defend the arsenals while the guerrillas have \(g\) units with which they can attack. Since \(p\geq 2g\) implies the police always win and \(p\leq g\) implies the guerrillas always have a winning strategy, we generally are sovling games where $\(g \leq p < 2g\)$ Common scenarios would have us analyzing 5P vs 3G, 7P vs 5G, or 8P vs 7G. We will conduct an emprical approach to solving this game, so the initiailization code below will be helpful.

Intial Values#

We first initialize the values we need:

  • \(p\) : number of police units

  • \(g\) : number of guerrilla units

  • strat : an arrays of strategies listed by larger number of the two groups of units

  • index : the column or row numbers in the results table

p = 8
g = 7

Dictionaries#

Below, we create a function that creates player dictionaries to keep track of row-column indexes for the results table and strategies listed by integer value.

def create_player_dict(name, anchor_val):
    """
    Creates a dictionary containing player info, strategy sets, and index sets.
    """
    strat = np.arange(math.ceil(anchor_val / 2), anchor_val + 1)
    index = np.arange(1, len(strat) + 1)
    return {
        'player': name,
        'index': index,
        'strat': strat
    }

# Creating the dictionaries
P = create_player_dict('P', p)
G = create_player_dict('G', g)

# Displaying the results
print(P)
print(G)

Words#

The strategies we use can be indexed by integer, say, 7. However, the following function will covert integers to the appropriate words which, for the preceding case, would be “seven”.

def words(num_array):

    return [engine.number_to_words(n) for n in num_array]

def word(n):
   return engine.number_to_words(n).replace("-", "_")

def play(p_max, g_max, p_strat, g_strat):
   A = p_strat ; B = p_max - A ;
   res = ()
   for k in range(90):
      a = np.random.choice(make_array( g_strat, g_max - g_strat ) ) ; b = g_strat - a ;
      if ( ( a > A ) or ( b > B ) ):
         v = 0
      else:
            v = 1
      res = np.append(res, v)
   temp = ( sum(res) + 5 ) / 100
   t = round(temp,3)
   return t
